#### Download use and configure 8n8
```
docker run -d --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  --restart always \
  n8nio/n8n
```
this will made 8n8 can be accesed in localhost:5678  
and safe all your work in a folder called .n8n in your Debian home directory so you don't lose your workflows.

#### 8n8 runing right ?
to check it type  
`docker ps`
u will see docker there  

#### check id 8n8 ready
`docker logs n8n --tail 20`

#### Star stop 8n8
`docker stop n8n`
`docker start n8n`

### if Error
- stop and remove docker  
```
docker stop n8n
docker rm n8n
```
  
- fix 8n8 folder permision  
`sudo chmod -R 777 ~/.n8n`

- start 8n8 again
```
docker run -d --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  --restart always \
  n8nio/n8n
```
  
- check logs  
`docker logs n8n --tail 10`  
If it worked: You should see something like n8n ready on 0.0.0.0, port 5678 or Editor is now available.  
If it failed: It will still show that EACCES error.
